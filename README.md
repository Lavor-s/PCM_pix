# PCM_pix

## Структура модулей (`pcm_pix/`)

### Группа численного эксперимента
- `main.ipynb` - cценарий численного эксперимента
- `run.py` — структура проекта и управление путями
- `solution.py` — результаты численного эксперимента

### Группа обучения суррогатов
- `features.py` — подготовка датасета суррогатных нейросетей 
- `nn_surrogate.py` — обучение и загрузка суррогатов
- `metrics.py` — оценка качества суррогатов (MAE/RMSE для амплитуд и фаз)

### Группа оптимизации
- `optimize.py` — оптимизация геометрии
- `hyperopt.py` — подбор гиперпараметров оптимизаторов:

### Группа ввода данных и графических ноутбуков
- `plots.ipynb` — основной графический ноутбук
- `io.py` — загрузка материалов (GST, Sb2Se3)
- `lum_tables.py` — табличные данные из Lumerical

### Фабрикация
- `gds.ipynb` — ноутбук‑обёртка над `gds.py` для подготовки файлов под фаб.
- `gds.py` — экспорт GDS и текстовых описаний (a/d/b + meta) для фабрикации.
